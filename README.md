# BelgradeQuest

BelgradeQuest je mobilna aplikacija razvijena u okviru predmeta *Razvoj mobilnih aplikacija i servisa*. Aplikacija omogućava korisnicima da istražuju grad kroz interaktivne zadatke (questove) vezane za različite lokacije.

## Funkcionalnosti

- Prikaz mapa i lokacija
- Sistem questova koje verifikuje AI
- Praćenje korisničkog napretka i osvojenih poena
- Korisnički profil
- Prijateljsko takmičenje
- Komunikacija sa backend Supabase servisom
